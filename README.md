# POC-Shell-Scraper

## Flags

<table>
  <tr>
    <td>-u &lturl></td>
    <td>Page URL (mandatory flag)</td>
  </tr>
  <tr>
    <td>-t &lttableid></td>
    <td>Table ID (mandatory flag)</td>
  </tr>
  <tr>
    <td>-f &ltfile></td>
    <td>Output file (mandatory flag)</td>
  </tr>
  <tr>
    <td>-h &ltfile></td>
    <td>Print helper message</td>
  </tr>
</table>

## Example
./scraper -u "https://gtfobins.github.io" -t "bin-table" -f "db"
